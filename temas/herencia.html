Gregor Mendel: El padre de la genética

Gregor Mendel (1822-1884) fue un monje que descubrió cómo se pasan las características de padres a hijos cruzando plantas de guisante (arvejas). Observó cómo se heredaban cosas como el color de la flor o la forma de la semilla.

Conceptos que debes conocer

  • Gen: Unidad de herencia. Un "pedacito de ADN" que determina una característica.
  • Alelo: Cada "versión" de un gen. Por ejemplo, el gen del color de ojos tiene un alelo para ojos marrones y otro para ojos azules.
  • Genotipo: La combinación de alelos que tienes "por dentro" (ejemplo: AA, Aa, aa).
  • Fenotipo: Lo que se ve "por fuera" (ojos marrones, pelo liso, etc.).

Dominante y Recesivo

  • Alelo dominante (MAYÚSCULA): Es el que "gana" y se muestra. Ejemplo: ojos marrones (A) domina sobre ojos azules (a).
  • Alelo recesivo (minúscula): Solo se muestra si no hay uno dominante. Necesitas dos alelos recesivos (aa) para tener ojos azules.

Ejemplo práctico

Cuadro de Punnett

Si un papá tiene AA (ojos marrones) y una mamá tiene aa (ojos azules), todos los hijos serán Aa (ojos marrones). Pero llevan el alelo de ojos azules "escondido".

¿Sabías que...?

El color de ojos no depende de un solo gen. En realidad hay varios genes involucrados, pero el modelo de Mendel es una buena forma de empezar a entender la herencia.
